# -gyomu-tools
業務効率化ツール集

## ツール一覧
- 見積書・請求書作成（`estimate.html`）
- 経費管理
- 現場管理

## 見積書・請求書作成（estimate.html）
Crastoreの御見積テンプレ／御請求テンプレをそのまま再現したツールです。
ロゴ・QRコード・書体・レイアウトはテンプレートに合わせています。

### 使い方
1. `estimate.html` をブラウザで開く
2. 上部の「見積書」「請求書」ボタンで切り替え
3. 各項目（宛名・No.・現場住所・工事内容・明細など）を直接クリックして入力
   - 数量×単価を入れると金額・小計・税額（10%）・合計を自動計算
   - 「一式」など数量無しの行は、金額欄に直接金額を入力すればOK
4. 「印刷 / PDF保存」ボタン →「PDFとして保存」で出力（A4）

### 工事内容をまとめて入れる場合
`estimate.html` の `<script>` 内、`★ ここに工事内容を入力 ★` のコメント下にある
`data` オブジェクトを書き換えるだけで、見積書・請求書の両方に反映されます。

```js
const data = {
  date:       "2026/06/01",      // 発行日
  clientName: "株式会社〇〇",     // 宛名（御中の前）
  docNo:      "2026-001",        // 見積書No. / 請求書No.
  siteAddr:   "大阪市〇〇1-2-3",  // 現場住所
  workName:   "店舗内装工事",     // 工事内容
  validUntil: "発行日より30日間", // 有効期限（見積書のみ）
  remark:     "",                // 備考欄
  taxRate:    10,                // 消費税率(%)
  items: [
    { detail: "軽量鉄骨下地組",  qty: 1,   unit: "式", price: 180000 },
    { detail: "ボード張り",      qty: 120, unit: "㎡", price: 1500 },
    { detail: "電気工事 一式",   unit: "式", amount: 95000 }, // 金額直接指定
  ]
};
```

ロゴとQRコードは `assets/` に保存し、`estimate.html` 内にbase64で埋め込み済みのため、
単体ファイルでもどこでも正しく表示・印刷できます。

### スマホ対応
画面幅に合わせて用紙(A4)を自動縮小表示するため、スマホ・タブレットでもそのまま開けます。
各項目はタップして入力でき、「印刷 / PDF保存」でスマホからもA4のPDFとして保存できます
（iPhoneはSafariの共有→「プリント」→ピンチで開いてPDF保存、AndroidはChromeの印刷→PDF保存）。

## 経費管理
日々の経費を記録・集計し、月次の支出状況を把握するためのツールです。
領収書情報の入力や、費目ごとの集計、月次レポートの作成をサポートします。
